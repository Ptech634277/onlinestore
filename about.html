How to edit this e‑commerce site

This template is built with plain HTML, Tailwind CSS (CDN) and vanilla JavaScript. You can open the project in VS Code and customize it easily.

  1. Change brand name & logo: Search for NeoStore text in all HTML files and replace with your store name.
  2. Update colors: Open css/style.css and adjust root color variables or add your own Tailwind classes.
  3. Edit products: Open js/products-data.js. Update the product list (name, price, category, image URL, etc.).
  4. Connect backend: Use this as the front‑end and connect with your API (Node, PHP, Python, Firebase, etc.).
  5. Deploy: Upload all files to any hosting: cPanel, Netlify, Vercel, GitHub Pages or your own server.

Tip for developers

The cart uses localStorage so it works without backend. Product details are loaded via productId from the URL query (example: product.html?id=1).

You can replace the dummy checkout and login with your real authentication and payment logic.
